Etcd storage adapter for [Keyv](https://github.com/jaredwray/keyv).

## Install

```shell
npm install --save keyv @keyv/etcd
```

## Usage

```js
const Keyv = require('keyv');

const keyv = new Keyv('etcd://localhost:2379');
keyv.on('error', handleConnectionError);
```
